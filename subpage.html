Some Linux Commands!

Command Explaination
cd Goes to the home directory
ls Goes to the home directory
ps Shows currently active processes
Al- Returns to the last directory
sudo Allows the user to log in as someone else
time Summarizes system resource usage
Yes Please Spams “please” over and over
w Shows system upload time, load averages and other information
locate Finds a file. Works better than the “find”
